Provides the class for a general optical element, which serves to connect the “lab-frame”, in which the light rays are traced, to the proper “optic” coordinate frame in which an optic is defined. So this class is what serves to "align" the optics in lab-space, and it provides methods to modify its alignment.

The optic can be a Mirror-object or a Mask-object.

 28class OpticalElement(ABC):
 29    """
 30    An optical element, can be either a Mirror or a Mask. In the future, one could add Gratings, DispersiveMedium etc...
 31
 32    Attributes: TODO update
 33    ----------
 34        position : np.ndarray
 35            Coordinate vector of the optical element’s center point in the lab frame.
 36            What this center is, depends on the 'type', but it is generally the center of symmetry
 37            of the Support of the optic. It is marked by the point 'P' in the drawings in the
 38            documentation of the Mirror-classes for example.
 39        
 40        orientation: np.quaternion
 41            Orientation of the optic in the lab reference frame. From this, we calculate
 42            the normal, the majoraxis and any other interesting vectors that are of 
 43            interest for a specific OpticalElement subclass. 
 44
 45        normal : np.ndarray
 46            Lab-frame vector pointing against the direction that would be considered
 47            as normal incidence on the optical element.
 48
 49        majoraxis : np.ndarray
 50            Lab-frame vector of another distinguished axis of non-rotationally symmetric optics,
 51            like the major axes of toroidal/elliptical mirrors or the off-axis direction of
 52            off-axis parabolas. This fixes the optical element’s rotation about 'normal'.
 53            It is required to be perpendicular to 'normal', and is usually the x-axis in the
 54            optic's proper coordinate frame.
 55
 56            What this 'majoraxis' is, e.g. for the different kinds of mirrors, is illustrated
 57            in the documentation of the Mirror-classes.
 58
 59            For pure p or s polarization, the light incidence plane should be the
 60            plane spanned by 'normal' and 'majoraxis', so that the incidence angle is varied
 61            by rotating the optical element around the cross product 'normal' x 'majoraxis'.
 62

102            self._q = NewOrientation.normalised()
103        else:

106    @property

108        return self.basepoint_ref + self.r

112        if type(NewPosition) == np.ndarray and len(NewPosition) == 3:
113            self._r = NewPosition - self.basepoint_ref
114        else:

118    ## Start of special points and axes definitions

121        """The centre of the optical element in the lab frame."""
122        return mgeo.GlobalPointCoord(self, self.centre_ref)
123
124    @centre.setter
125    def centre(self, NewPosition):
126        """We assume no change in the orientation of the optic when the centre is moved."""
127        if type(NewPosition) == np.ndarray and len(NewPosition) == 3:
128            self._r += NewPosition - self.centre
129            self.centre_ref = mgeo.LocalPointCoord(self, NewPosition)
130    
131    @property
132    def support_normal(self):
133        return (self.q*np.quaternion(0.0,*self.support_normal_ref)*self.q.conj()).imag

137        return (self.q*np.quaternion(0.0,*self.majoraxis_ref)*self.q.conj()).imag

144    def rotate_pitch_by(self, angle):
145        """
146        Pitch rotation, i.e. rotates the optical element about the axis ('normal' x 'majoraxis'), by the
147        given angle.
148        If the plane spanned by 'normal' and 'majoraxis' is the incidence plane (normally the case
149        in a "clean alignment" situation for pure p or s polarization), then this is simply a modificaiton
150        of the incidence angle by "angle". But in general, if the optical element has some odd orientation,
151        there is not a direct correspondence.
152
153        Parameters
154        ----------
155            angle : float
156                Rotation angle in *degrees*.
157        """
158        rotation_axis = np.cross(self.support_normal, self.majoraxis)
159        self.q = mgeo.QRotationAroundAxis(rotation_axis, np.deg2rad(angle))*self.q
160

163        Roll rotation, i.e. rotates the optical element about its 'majoraxis' by the given angle.
164
165        Parameters
166        ----------
167            angle : float
168                Rotation angle in *degrees*.
169        """
170
171        self.q = mgeo.QRotationAroundAxis(self.majoraxis, np.deg2rad(angle))*self.q

222
223        Parameters

225            distance : float
226                Shift distance in mm.
227        """
228        self.position = self.position +  distance * mgeo.Normalize(np.cross(self.support_normal, self.majoraxis))
